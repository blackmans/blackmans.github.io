随便说说line-height

  • css
  • line-height
  • vertical-align

line-height语法

line-height：normal | length | percentage | number
默认值：normal
适用于：所有元素
继承性：有

  • normal: 默认合理的值
  • length: 长度值，不可以使用负值
  • percentage: 计算与字体大小的乘积
  • number: 同样是计算与字体大小的乘积
  • 注意：与的区别在于，在子级集成祖先级的时候，前者是先与祖先级计算在让子级继承；而后者则是先让子级集成，再与子级字体大小进行自己算；

与line-height相关的四个框或盒子（box）

1.content area（内容区域）


就是文字撑开的范围，是有font-size决定的；

2.inline box（行内框）


inline box 的高度是由行高决定的，这里值得注意的是，inline元素和inline-block表现上是不一样的；同样都设置同样的行高，并且都设置背景颜色，前者不会显示与行高一样的背景范围，但是后者则可以，这是因为inline元素不支持宽高的原因。

3.line box（行框）


line box 是由inline box组成的，高度也是有inline box决定的；超出容器范围时line box会自动折行，形成多个line box；

4.contain box


contain box是由line box做成的；

line-height与四种框的关系

1.行高的表现

行高是基线到基线的距离，下图中也体现出了行距；

2.行内框

行内框实际是，行高减去文字的高度/2；
